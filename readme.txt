Analysis of sales order report with various important metrics, that are present behavior of woocommerce store sales performance with the uses of varis charts and key data.

== Description ==
The sales order report plugin for woocommerce store is helpsful to an analysis of business store sales, given the reports and charts with different metrics. report and chart present with different date range options. likes, Today, Last 7 Days, Last 30 Days, This Month, Last Month, Last 3 Months.

The performance of the sales order report plugin is the ability to fetch data is very fast with the presence of multiple charts and important sales various metrics data.

Sales order key data will option to download the CSV file using full order data like Order ID, Date, Product Name, Quantity, Line amount, Shipping, TAX.


FEATURES:

*   Sales order analysis with different date range options like Today, Last 7 Days, Last 30 Days, This Month, Last Month, Last 3 Months
*   Represents the woocommerce store important metrics of sales order performance with good visual UI
*   Different charts are given visibility of easy to understanding to analyze sales order. 
*   Give sale order summary with various date range
*   Download the sales order data with Order ID, Date, Product Name, Quantity, Line amount, Shipping, TAX.
